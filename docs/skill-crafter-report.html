skill crafter

Different models need different skills.

A coding SKILL file is a list of behavioral statements. Some are load-bearing for a given model; others tell it to do what it already does. Skill crafter probes each statement per model and keeps only the ones that change behavior.

For every atomic claim, the judge model (Qwen3.5 (122B-A10B; 262k ctx; Q5_K_XL)) authors a disguised question and a rubric. The target answers it with the whole skill loaded and with nothing, 3× each; the judge decides whether the skill changed the answer. keep = it did, drop = the model already behaves that way with no skill at all.
