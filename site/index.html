ACP agent · Zed ⇄ any local LLM

A holder for your code.

codehalter is a local coding agent that connects Zed to any OpenAI-compatible model. It holds your code, your context, and every gated step — plan, execute, verify, document — entirely on your machine.

$ go install github.com/tbocek/codehalter@latest

// the loop

Four gated steps. Nothing runs unwatched.

Every task moves through the same disciplined pipeline. The agent proposes; the holder keeps each step in scope before it touches your repo.

01

Plan

Breaks the task into a reviewable list of concrete steps before writing a single line.

02

Execute

Works one step at a time inside Zed, editing files through the ACP bridge.

03

Verify

Builds, runs tests, and checks its own output before moving on. Failures halt the loop.

04

Document

Writes down what changed and why, so the next session — human or agent — has context.

// on your machine

// built for control

Designed to be held, not trusted blindly.

Every primitive assumes you want to stay in the loop.

Scoped permissions

The agent only touches files you've put in scope. Everything else is off-limits by default.

Reviewable diffs

Changes land as diffs you approve in Zed. Nothing is committed without a green light.

Halts on failure

A failed build or test stops the loop cold. No charging ahead on a broken state.
